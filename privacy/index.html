Website privacy

The website is a static product catalog.

ariavisionapps.com is a static website used to present Aria Vision Windows apps, screenshots, product descriptions, related app links, and Microsoft Store links.

The website itself does not collect personal data, does not provide user accounts or login, does not use analytics, does not set cookies, and does not store visitor information in a backend database.

The site is hosted as static HTML, CSS, JavaScript, images, XML, and TXT files. It does not include a backend service, tracking script, advertising pixel, or server-side user profile system.

At a glance

What this website does not use

  • No website login or account system
  • No website analytics or tracking cookies
  • No advertising pixels
  • No backend storage for website visitors
  • No collection of personal data through the website

Windows apps

Individual apps may have their own privacy details.

Aria Vision publishes multiple Windows apps on Microsoft Store. Privacy behavior can vary by product depending on what the app does, the features it includes, and any privacy policy linked from its Microsoft Store listing.

Before installing or using an app, review the app detail page on this website and the Microsoft Store listing for product-specific information when available.

If you have a question about a specific app, include the app name when contacting support at sn.moosavi@gmail.com.
